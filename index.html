# Getting Started

# Quick Start

The fastest way to start solving AoC problems is just by installing this package.

npm i aoct

WARNING

For the moment, online data is not availible. So you will need to have puzzles data locally. See how here.

To simplify your run command, add following to your package.json:

"scripts": {
  "start": "npx aoct start"
},

Select your day and you're ready to go! ðŸŽ‰

npm start day1

# TypeScript

If you are interested in solving problems with TypeScript, you will need aoct and ts-node.

npm i aoct ts-node

Create aoct.json specifying TypeScript compiler. See more here.

{
  "compiler": "ts"
}
